## Access Your Dashboard

Navigate to `app.gleef.eu/dashboard` and log in with your Gleef account. You'll see:

- **Draft Translations** - Pending items awaiting review
- **Changes to Review** - Modifications ready for approval

## Start Translating

<img width="914" src="/images/studio/translation-table.png" alt="translation table screenshot"/>

1. **Click "View my translations"** to access the translation table
2. **View other languages** using the "+" button in the header
3. **Click any cell** to edit translation content
4. **Changes save automatically** as you work
5. **Publish or ask for a review** with the top-right button

## Review and Publish

<img width="2206" src="/images/studio/review-figma.png" alt="review page screenshot"/>

1. **Navigate to Review** to see pending changes
2. **Expand changeset cards** to review modifications
3. **Approve or reject** individual translations
4. **Use the Publish button** to deploy approved changes

## Set Up Translation Rules

<img width="2204" src="/images/studio/rules.png" alt="translation rules screenshot"/>

1. **Go to Translation Rules** section
2. **Click "Create New Rule"** to customize AI behavior
3. **Define tone, formatting, and style guidelines**
4. **Rules automatically apply** to new translations

## Next Steps

- Use **Search** to find specific translations or rules
- Connect **Figma designs** for visual context
- Set up **API keys** for CLI integration
