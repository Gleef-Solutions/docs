## Overview
This section will guide you through the process of installing and setting up our Figma Plugin in your project.

<Steps>
  <Step title="Install the plugin">
    Install the plugin from the Figma community, directly in your project.
    <Info>
      You can find the plugin [here](http://figma.gleef.eu).
    </Info>
  </Step>
  <Step title="Sign up">
    Sign up for a Gleef account. You can do this directly in the plugin.

    <Warning>Currently, you can't join someone's company to share the same database on your own. If you want to do so, please [contact us](contact@gleef.eu)</Warning>
  </Step>
  <Step title="Select language">
    You must select the exact language you're using when working on Figma, so that Gleef can correctly match your existing keys with what you're designing.

    <Warning>Currently you can only manage one language.</Warning>
  </Step>
  <Step title="Add your existng keys">
    It's an important step if you want to use our product. We highly recommend you to add your existing keys to our database.
    Doing so will enable our AI to better format keys, retrieve existing keys, avoid duplication and fit your patterns.
    <Info>
      You can find more information about how to add your existing keys in the [documentation](/docs).
    </Info>
  </Step>
  <Step title="Start Generating keys">
    Select a frame in your Figma project with our Plugin opens, and click on generate keys. It might takes 10 to 60 seconds to generate the keys.
    <Info>
      You can find more information about how to generate keys in the [documentation](/docs).
    </Info>
  </Step>
  <Step title="Manage keys">
    With the current frame, you can now see all keys that were retrieved or generated. You can easily:
    - Display keys
    - Click on each key to focus the element in the frame
    - Edit keys
    - Remove keys
    - Re-generate keys
    - Export keys to a JSON, CSV or XML file
  </Step>
  <Step title="Continue on!">
    Perfect, you're ready to continue generating more keys.
  </Step>
</Steps>
