## Installation
Follow these simple steps to install our plugin and get it work on your Figma.

<Steps titleSize="h3">
  <Step title="Download the plugin">
    Install the plugin from the Figma community, directly in your project.
    
    You can find the plugin [here](http://figma.gleef.eu).
  </Step>
  <Step title="Launch the plugin directly in each project">
    Once installed, you can easily open the Plugin wherever you need to create or manage localization keys.
  </Step>
</Steps>
